Páxina principal

Esta páxina principal serve como aloxamento de todas as demais xanelas que son mostradas unha vez é pulsado un dos botóns que indican cada unha das opcións de xestión.


Coma se pode observar na imaxe superior, a pantalla principal componse de :

  1. Cómics: Ao pulsar este botón é mostrase a xestión sobre os cómics dispoñibles. Preme para máis información
  2. Autores: Pulsando sobre este botón o usuario accede á pagina de xestión sobre os autores de cómics.
  3. Buscador:Desde a pantalla de procura é posible realizar diferentes consultas sobre os datos almacenados na BD.
  4. Informe de cómics:É posible a través deste panel consultar e mostrar a información dos cómics contida no xestor/BD resumida en documentos útiles para o usuario .
  5. Informe de autores: Este panel permite consultar e mostrar a información dos autores en documentos útiles para o usuario .
